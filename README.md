Консольная система студенческих оценок

Описание проекта

Консольное приложение на Java для управления студентами, предметами и оценками. Проект разработан с использованием принципов ООП и коллекций Java.

Возможности

* Добавление студентов
* Добавление предметов
* Добавление оценок
* Расчёт среднего балла по предмету
* Расчёт GPA студента
* Просмотр предметов студента
* Просмотр всех оценок по предмету
* Аналитика успеваемости

Используемые технологии

* Java Core
* ArrayList
* HashMap
* ООП
* Scanner

Структура проекта

src/
├── model/
├── service/
└── main/

Запуск программы

1. Скачать или клонировать проект:
    git clone <ссылка_на_репозиторий>
2. Открыть проект в IntelliJ IDEA или другой IDE.
3. Запустить файл Main.java.
4. Использовать меню в консоли для работы с системой.
